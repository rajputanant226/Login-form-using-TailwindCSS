#  Responsive Login Form using Tailwind CSS
### Overview
This project is a simple yet modern responsive login form built with Tailwind CSS, showcasing the power of its utility-first styling approach.
It demonstrates how to quickly create a clean, centered login interface that looks great on all screen sizes — without writing a single line of custom CSS!

### Features
# Responsive Layout: Works seamlessly on mobile, tablet, and desktop screens
# Utility-First Styling: Uses Tailwind’s pre-built utility classes for design
# Modern UI Design: Clean inputs, hover effects, and soft shadows
# Dark Mode Ready: Easily customizable for light/dark themes
# Minimal & Reusable: No extra CSS — pure Tailwind utilities
# Technologies Used
HTML5 – Markup structure
Tailwind CSS – Styling and layout
(Optional) Vite / CDN Setup – For quick configuration
